Files that are inside local/ftbutilities/:

  • all_permissions.html - Usually cut-off fancy page for all permissions FTBU can use
  • all_permissions_full_list.txt - Plaintext list of all permission nodes
  • config.json - General FTBU configuration
  • player_ranks.json - Stores rank of players by UUID
  • ranks.json - Stores ranks
  • server_badges.json - Badge configuration
  • all_configs.html - All available configuration nodes for FTBU
