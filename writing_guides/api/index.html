Available files

  • types.json - Lists supported guide types
    Key Type Description
    id String Lowercase ID
    title String Title
    title_p String Title (Plural)
  • all.json - Lists all available guides. See data.json page
  • mod.json - Example type-specific json. type key is missing from type-specific jsons.
  • style.css - CSS file that controls how Guides look like.
  • custom_elemennts.js - JavaScript that processes HTML on client side (mostly used for fixing image paths and custom elements).

Updating

The API / Guides update every time something is pushed to the FTBGuidesWeb repository to minimize resource usage. All files are static.
